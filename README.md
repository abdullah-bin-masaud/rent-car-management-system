## How to Run Locally
1. Install XAMPP
2. Place project Files in htdocs folder in XAMP
3. Import database from /database/rent_car_db.sql
4. Start Apache & MySQL on XAMP
5. Make Database on localhost PhpMyAdmin
5. Open browser and visit:
   http://localhost/CEP
